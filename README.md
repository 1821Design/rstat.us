rstat.us
========

rstat.us is a microblogging site built on top of the [ostatus
protocol](http://status.net/wiki/OStatus).

Running your own
----------------

Just do this:

    $ git clone https://github.com/hotsh/rstat.us.git
    $ cd rstat.us
    $ heroku create
    $ git push heroku master

Bam!
